# Darkpager Theme

**Author:** sinXne0  
**Version:** 1.0  
**Firmware Version:** 1.04

## Description

Black background with yellow text and a custom lock screen, dashboard, and boot animation image.

## Installation

1. Copy the `darkpager` theme directory to `/root/themes` on your Pager
1. Select the theme from the Pager settings

## Credits

Theme structure based on the Matrix theme.
